# 🏫 YP 학생회(선도부) 상벌점 및 일정 관리 시스템

고등학교 학생회 및 선도부(YP) 학생들의 활동 일정, 출석, 그리고 전교생 상벌점 부여를 효율적으로 관리하기 위해 개발된 **서버리스(Serverless) 웹 애플리케이션**입니다. 

구글 시트(Google Sheets)를 데이터베이스로 활용하고 구글 앱스 스크립트(GAS)를 백엔드 API로 사용하여, 별도의 서버 유지비용 없이 무료로 학교 현장에 즉시 도입할 수 있습니다.

## ✨ 주요 기능 (Key Features)

* **🔐 직책별 권한 제어 (RBAC)**
  * 관리자(교사), 선도부장, 선도차장, YP학생 등 직책에 따른 메뉴 및 기능 접근 권한 분리.
* **⚡ 초고속 실시간 학생 검색**
  * 전교생 데이터를 브라우저 메모리에 캐싱하여 딜레이 없는 실시간 검색 지원.
* **📝 상벌점 부여 및 중복 방지**
  * 개별 및 다중(일괄) 상벌점 부여 지원.
  * 동일 날짜, 동일 항목, 동일 학생에 대한 중복 입력 자동 차단.
* **📅 주간 일정 관리 및 엑셀 연동**
  * 달력 형태의 직관적인 주간 YP 일정표 제공.
  * 엑셀(Excel) 양식 다운로드 및 업로드를 통한 대량 일정 등록 지원 (관리자 전용).
  * 달력 칸을 클릭하여 즉시 담당자를 수정하는 실시간 편집 기능.
* **⏰ 시간 기반 자동 출석 체크**
  * 본인의 담당 시간(-5분 ~ +5분)에 로그인 시 자동 출석 인정.
  * 해당 시간 외 출석 시 10자 이상 사유 입력 강제화.
* **📊 맞춤형 대시보드**
  * **학생용:** 나의 일정, 내가 부여한 상벌점 내역 조회.
  * **관리자용:** 일일 출석 현황, 전체 상벌점 부여 누적 기록 조회.

## 🛠️ 기술 스택 (Tech Stack)

* **Frontend:** HTML5, CSS3, Vanilla JavaScript, Bootstrap 5, SweetAlert2, SheetJS
* **Backend:** Google Apps Script (REST API 화)
* **Database:** Google Sheets

## 🚀 설치 및 세팅 가이드 (Getting Started)

이 시스템을 본인의 학교에 도입하려면 프론트엔드와 백엔드를 각각 세팅해야 합니다.

### Step 1. 구글 시트 데이터베이스 세팅 (Backend)
1. 아래 링크를 클릭하여 데이터베이스 템플릿(샘플 시트)을 본인의 구글 드라이브로 복사합니다.
   👉 **[YP 학생회 시스템 샘플 시트 복사하기]([여기에_copy로_끝나는_링크를_넣어주세요](https://docs.google.com/spreadsheets/d/1tV1D-vQW4c-vilRiv47AXN50nbAC0km86v5NxwZfbLM/copy))**
2. 메뉴에서 **[확장 프로그램] > [Apps Script]**를 엽니다.
3. 본 저장소의 `Code.gs`
4. 우측 상단 **[배포] > [새 배포]** 클릭 ➡️ 유형: 웹 앱, 실행 권한: 나, 액세스 권한: 모든 사용자 ➡️ 배포 후 생성된 **Web App URL**을 복사합니다.
 내용을 복사하여 붙여넣습니다...

### Step 2. 웹 화면 세팅 (Frontend)
1. 이 저장소의 코드를 Clone 하거나 다운로드합니다.
2. `config.js` 파일을 열고, Step 1에서 복사한 **Web App URL**을 붙여넣습니다.
3. `index.html`, `config.js`를 포함한 파일들을 Vercel, Netlify, 또는 GitHub Pages를 통해 무료로 웹 호스팅합니다.
